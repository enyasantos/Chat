## Run application
#### Run server
```
git clone https://github.com/enyasantos/chat-nodejs.git
cd server
yarn install or npm install
yarn start:dev or npm start:dev
```
#### Run frontend
```
git clone https://github.com/enyasantos/chat-nodejs.git
cd chat
yarn install or npm install
yarn start or npm start
```
